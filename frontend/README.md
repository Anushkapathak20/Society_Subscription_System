# Society Subscription System - Frontend

This is the Next.js frontend for the Society Subscription System.

For the full project documentation, including setup instructions for both frontend and backend, please refer to the **[root README.md](../README.md)**.

## 🚀 Quick Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
